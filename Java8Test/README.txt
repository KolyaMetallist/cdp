To launch the application type: java -jar textAnalyzer.jar and needed option parameters.

Options:
  -a, --approach
     Approach: Java8 or NonJava8
     Default: Java8
* -i, --input
     Input file
  -p, --parallel
     Parallel
     Default: false
* -t, --task
     Process task: frequency, length or duplicates
 --help
	 Help
	 
Required options are the input file and the task.
